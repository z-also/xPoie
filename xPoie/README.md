# Welcome Page Design And Implement

## 背景
我正在开发一款 macOS 的原生应用，主要功能包括
- 任务 todo 管理，for 用户日常和生活的工作效率
- 无限画布：可以拖拽生成各种卡片，包括 note、ai 生成图片、长文档 等等
- 一个随处可唤起的 ai 助手，帮助创建任务、搜索网络、响应指令
- 一个便捷的全局窗口，可以浮动在屏幕常驻 

概括来说，定位是给个人用的所有工作、创作、记录、效率 工具，现在需要实现一个启动的 welcome页面。你帮我设计这个页面，也给出相关的文案。
比如左右布局，左边上下是 logo、slogan、sign in with apple，右边是一些产品功能的轮播图或者视频。
